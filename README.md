# Playground

Playground to test out different Smart Contract and dApp ideas.

## Commands

```sh
. ./.env.sh

nix-shell

dapp build

dapp debug [--rpc]

dapp test [-m <string>] [-v] [--rpc] [--replay <string>]

npm run format
```
